# homebrew-swiftforge

Homebrew tap for [CodeForge](https://github.com/sachin6174/SwiftForge) (formerly SwiftForge) — a native SwiftUI app for practicing DSA and Swift/iOS interview questions offline.

## Install

```bash
brew tap sachin6174/swiftforge
brew install --cask codeforge
```

## Update

```bash
brew upgrade --cask codeforge
```

## Uninstall

```bash
brew uninstall --cask codeforge
```

> Upgrading from SwiftForge? The old `swiftforge` cask still works and now installs the same CodeForge 2.0.0 app, but new installs should use `codeforge`.
