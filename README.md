# homebrew-swiftforge

Homebrew tap for [SwiftForge](https://github.com/sachin6174/SwiftForge) — a native SwiftUI app for practicing DSA and Swift/iOS interview questions offline.

## Install

```bash
brew tap sachin6174/swiftforge
brew install --cask swiftforge
```

## Update

```bash
brew upgrade --cask swiftforge
```

## Uninstall

```bash
brew uninstall --cask swiftforge
```
